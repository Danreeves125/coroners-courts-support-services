# Divi-Child-Theme

# Project Title

A Child Theme for the Divi WordPress Theme built by Elegant Themes

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

The Divi WordPress theme must be installed on the site in order for this child theme to work.

## Built With

* [Options Framework Theme](https://github.com/devinsays/options-framework-theme) - The Additional Options attached to the theme (Dentons Digital Options)
* [Font Awesome](http://fontawesome.io/) - Iconography
* [Divi WordPress Theme](https://www.elegantthemes.com/) - WordPress Theme
